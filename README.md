# Sabithra M - Portfolio

A professional portfolio website showcasing my skills, projects, and experience.

## Features

- Responsive design
- Multiple pages with smooth navigation
- Professional layout
- Contact form
- Social media links

## Pages

- **Home**: Introduction and hero section
- **About**: Personal information, photo, and education
- **Experience**: Internships, achievements, and coding profiles
- **Projects**: Showcase of personal projects
- **Skills**: Technical skills and certifications
- **Contact**: Contact information and form

## Setup

1. Clone or download the repository
2. Open the files in your preferred code editor
3. Customize the content as needed
4. Upload your profile photo to `images/profile.jpg`
5. Upload your project screenshots to `images/`:
   - `fraud-detection.jpg`
   - `diet-plan.jpg`
   - `multi-vendor.jpg`
6. Upload your resume PDF to the repository root as `resume.pdf`
7. Update the social media links and contact information

## Deployment to Netlify

1. Create a free account on [Netlify](https://netlify.com)
2. Click "New site from Git" or "Deploy manually"
3. If deploying manually:
   - Drag and drop the entire portfolio folder
4. If using Git:
   - Connect your Git repository
   - Netlify will automatically build and deploy
5. Your site will be live at a Netlify subdomain (e.g., `https://yourname.netlify.app`)
